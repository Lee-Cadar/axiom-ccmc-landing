Method

How CCMC works

CCMC is an autonomous, geometry-aware Hamiltonian Monte Carlo system. It combines three components: a Riemannian leapfrog integrator on the torus T², a geometry-derived parameter selector (NREM), and an XGBoost model that learns optimal configurations from prior sampling experience.

01
Upload your geometry

Send your distribution parameters — Gaussian curvature K, torus radii R and r, and optional research overrides. No custom code required. The API accepts JSON; a web UI for file upload is in development.

POST /sample {"K": 0.25, "R": 3.0, "r": 1.0}
02
NREM derives parameters

NREM (Neighbourhood Radius for MCMC) computes optimal step size ε* and trajectory length L* from local curvature. XGBoost checks 37,537 prior experiments for similar configurations. No manual tuning.

ε* = c_ε / √ECR    L* = c_L · √CLS
03
JAX GPU executes

The generalised leapfrog integrator, JIT-compiled via JAX lax.scan, parallelised with vmap. Runs on GPU. You receive a job_id immediately. Results arrive async (poll or webhook).

GET /results/{job_id} → ess, r_hat, samples
04
Full diagnostics returned

ESS, R̂, acceptance rate, NREM parameters used, convergence verdict, data provenance (live vs theoretical), and stability constraint verification. Every output is traceable to its mathematical origin.

{"convergence": "PASS", "data_source": "live"}

g(u,v) = diag[(R + r·cos v)², r²]
Gaussian curvature
K(u,v) = cos v / [r(R + r·cos v)]

ε* = c_ε · (1 / √ECR)
NREM trajectory length

Stability constraint

Mathematical foundation

100% grounded in Riemannian geometry

CCMC operates on the 2D torus T² = S¹ × S¹. The Riemannian metric tensor encodes local curvature at every point. The generalised leapfrog integrator incorporates Christoffel symbols — preserving volume and time-reversibility on the curved manifold.

No approximations. No flat-space assumptions. Every integration step respects the geometry of the space being sampled.

The API

Two endpoints are free and require no authentication. Full sampling and batch access require an API key, available via the waitlist.

GET /health Platform status: JAX GPU online/offline, queue depth Free
POST /nrem Geometry → optimal ε* and L*. No sampling performed. Free
POST /sample Full CCMC run. Returns job_id immediately (async). Beta Access
GET /results/{job_id} Poll for results: ESS, R̂, samples, diagnostics. Beta Access
POST /sample/batch Submit up to 20 configurations in one request. Beta Access
GET /experiments 37,537+ validated benchmark statistics. Beta Access
